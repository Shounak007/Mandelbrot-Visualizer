Click and drag to pan, scroll to zoom
Hover to update Julia set

How to Use

  • Switch between Mandelbrot, Julia, and Dual views using the buttons above
  • Adjust the maximum iterations to control detail level
  • Choose different color schemes for visualization
  • Click and drag on the Mandelbrot set to pan
  • Use the mouse wheel to zoom in/out
  • Hover over the Julia set to update its parameters
  • Use the Julia Lock checkbox to freeze the Julia set parameters
  • Click Reset View to return to the initial view
